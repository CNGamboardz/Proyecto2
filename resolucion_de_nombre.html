RESOLUCIÓN DE NOMBRES

Definición Resolución de Nombres Tipos de DNS

La resolución es un proceso que se da desde un nombre de dominio a una dirección IP, y se puede dividir en pasos donde son los siguientes:


  • Los usuarios ingresan el nombre de dominio en su navegador web.
  • El navegador envía una solicitud al servidor DNS.
  • El servidor DNS busca el nombre de dominio en su base de datos.
  • Si el nombre de dominio existe, se devuelve la dirección IP correspondiente.
  • Los navegadores utilizan direcciones IP para conectarse a sitios web.

    ESTÁNDARES Y PROTOCOLOS


    El Sistema de Nombres de Dominio (DNS) se rige por varios estándares y protocolos definidos
    en documentos RFC (solicitar comentarios):


  • RFC 881: Plan y Cronograma de Dominio - El Plan de Dominio y su Cronograma, el inicio oficial del plan de trabajo de conceptualización del DNS.
  • RFC 1034: Dominios de red: conceptos y capacidades: define los conceptos y funciones básicos de DNS.
  • RFC 1035:Dominios: Implementación y especificaciones: Define la implementación del protocolo DNS.
  • RFC 2671:Mecanismo de extensión DNS (EDNS0): define extensiones del protocolo DNS para admitir paquetes más grandes.
  • RFC 2845: Autenticación de transacciones DNS con claves (TSIG): define un mecanismo de autenticación para transacciones DNS mediante claves.
  • RFC 3225: Instrucciones para que los solucionadores admitan DNSSEC: indica a los solucionadores cómo indicar su compatibilidad con DNSSEC.
